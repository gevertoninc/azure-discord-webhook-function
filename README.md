# azure-discord-webhook-function

## Introdução

Este projeto é um webhook que envia eventos da Azure para o Discord

## ESLint

Para rodar o [ESLint](https://eslint.org), execute `npm run eslint`

## Prettier

Para rodar o [Prettier](https://prettier.io), execute `npm run prettier`

## Instalar dependências

Para instalar dependências, execute `npm i`

## Compilar TypeScript

Para compilar os arquivos TypeScript, execute `npm run build`

## Rodar localmente

Para rodar localmente, execute `npm run start`
